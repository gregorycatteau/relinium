# Code de conduite

Relinium favorise un environnement bienveillant, respectueux et inclusif.

## Nos engagements

- **Respect mutuel** : chaque personne est traitée avec dignité, quelles que soient ses compétences techniques.
- **Bienveillance** : on privilégie l'écoute et l'empathie dans nos échanges.
- **Pas d'attaques personnelles** : on critique les idées, pas les personnes.
- **Inclusion** : toutes les contributions légitimes sont bienvenues.
- **Transparence** : les désaccords se traitent via issues ou RFC, de manière publique et constructive.

## Comportements inacceptables

- Harcèlement sous toute forme
- Propos discriminatoires ou offensants
- Divulgation d'informations personnelles sans consentement
- Intimidation ou menaces
- Tout comportement qui nuit à un environnement sain et collaboratif

## Signalement

En cas de violation du code de conduite :
- Créer une **issue Security** (pour les problèmes graves)
- Contacter par email : **contact [at] relinium.io**

Tous les signalements seront traités avec sérieux et confidentialité.

## Conséquences

Les violations peuvent entraîner :
- Avertissement formel
- Bannissement temporaire ou permanent du projet
- Signalement aux plateformes concernées

## Portée

Ce code s'applique à tous les espaces du projet :
- Dépôt GitHub (issues, PR, discussions)
- Canaux de communication officiels
- Événements associés au projet
